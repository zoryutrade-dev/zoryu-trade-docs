# Zoryu Trade Docs

Documentation hub for **Zoryu Trade** — a non-custodial, execution-focused trading terminal for Solana.

This repository contains:
- Product overview & principles
- System architecture (data flow, execution flow)
- Module specifications (Terminal, AI Signals, Dashboard)
- API reference and contract alignment
- Security model and threat boundaries
- Developer guide and deployment notes
- Roadmap and current status

## Philosophy
Zoryu prioritizes:
- **Non-custodial control**
- **State-aware trading workflows**
- **Measurable outcomes**
- **Execution transparency**
