# Setup

This guide outlines the basic setup required to work with Zoryu Trade components.

## Prerequisites

- Node.js (v18+)
- npm / pnpm / yarn
- PostgreSQL-compatible database (for backend services)
- Solana wallet for execution testing

## Environment Variables

Zoryu services rely on environment-based configuration.

Typical variables include:
- database connection URL
- API keys for external services
- network configuration

Sensitive values are never committed to source control.
