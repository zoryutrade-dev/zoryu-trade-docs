# Quick Tour

This section describes the main modules and how they fit together.

## Trading Terminal

The Trading Terminal is the operational interface for monitoring and execution.

It provides:
- real-time market data (price, change, volume)
- charting for context
- order entry with explicit slippage control
- wallet connection and balance visibility

## Dashboard

The Dashboard provides a consolidated view of system state:
- active positions
- performance statistics
- recent history
- navigation to terminal and signals
